4xx: Client error
A client error indicates that the request cannot be completed because of an issue with the client, or the syntax of the request.

400 Bad Request
401 Unauthorized
403 Forbidden
404 Not Found
408 Request Timeout
429 Too Many Requests
440 Login Time-Out
444 No Response
451 Unavailable For Legal Reasons
